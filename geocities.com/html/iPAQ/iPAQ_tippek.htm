    Date & Time Registry Hack
    Marshall Burke emailed us this rather interesting registry hack that some of you might be interested in. You'll need a registry editor, so check this one out. Here's what Marshall suggests 
    "I was very annoyed when Microsoft removed this function (Date and Time) so I pulled out my old PPC and started to analyze the Registry and found the setting to allow it on the PocketPC operating system (I have tested on both PocketPC 2000 and PocketPC 2002). Use a Registry Editor and add the follows:
    Add a Binary Value to the following Section:

    \HKLM\Software\Microsoft\Shell
    Value Name: TBOpt
    Value Data: 13 00 00 00

    Do a soft reset and you will get both the time and date. The disadvantage is that you can't switch to the totally useless analog clock (it will show '1/1'). I believe what the above registry setting does is to decrease the font value of the time and date area to allow both items to be displayed."

    WinCE fájl szinkronizáció könyvtár beállítás
    Szerzõ: Lacó
    2001-12-13 13:56:38

    Az ActiveSync egyik igen hasznos szolgáltatása a fájlok szinkronizálása az asztali gépünk és PDA-nk között. Én ezt arra szoktam használni, hogy backup helyett szinkronizálom az adatokat, így esetleges probléma esetén, vagy egy új gép használatbavételekor nem kell a visszatöltéssel bajlódni, elég a szinkronizációt beindítani. A másik elony, hogy egy szinkronizáció lényegesen gyorsabban lezajlik mint egy teljes backup. Persze hátránya is van a módszernek, a szinkronizáció nem menti el a felinstallált programjainkat és azok beállításait, így érdemes a szinkronizáció mellett programok installálása után csinálni egy backupot is.
    A fájl szinkronizáció beállításainál nem lehet megadni sem az asztali sem pedig a hordozható gépen, hogy melyik könyvtárat szinkronizálja, az alapértelmezés pedig nem mindig megfelelo. Ennek ellenére van megoldás.

    Az asztali gépen a HKEY_CURRENT_USER/Software/Microsoft/Windows CE Services/Partners alatt található kódok közül ki kell választanunk a szóban forgó gépet, ezt a benne található Display Name kulcs alapján tehetjük meg és utána az alatta levo Sevices/Syncronization/Briefcase/Path kulccsal állíthatjuk be a szinkronizálás könyvtárát.

    A hordozható gépen a HKEY_LOCAL_MACHINE/Software/Microsoft/Windows CE Services/FileSyncPath kulcs segítségével lehet ugyanezt beállítani. Figyelni kell arra, hogy annak ellenére, hogy az asztali gépen is van egy kulcs a ami a PDA könyvtárát állítaná be, az nem muködik.

    Végül egy jó tanács: mielott ezeket a beállításokat megváltoztatnánk csináljunk egy másolatot az értékes fájljainkról, nehogy az átállítás után talált üres könyvtárat szinkronizálja rá a másik gépen meglévo dokumentumokra...
